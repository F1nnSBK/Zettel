Eine **Menge** ist eine grundlegende [[mathematische Struktur]] und eine Sammlung von bestimmten, wohlunterschiedenen [[Element|Elementen]].
Eine Menge kann endlich oder unendlich viele Elemente enthalten.

Ist ein Element $a$ Teil einer Menge $M$, so schreibt man: $$a \in M$$
Ist $a$ kein Element der Menge $M$, schreibt man: $$a \notin M$$
**Kardinalität**
Die **Mächtigkeit** (oder **Kardinalität**) eienr Menge gibt die Anzahl ihrer Elemente an. Sie wird mit Betragsstrichen $\vert M \vert$ notiert. Für die Menge $M = \{1,2,3,4,5\}$ ist die Mächtigkeit: $$\vert M \vert = 5$$
